## 🔍 نمای کلی پروژه

در دنیای ورزش حرفه‌ای، اتکا به «شهود» کافی نیست. هدف این فاز، پاسخ به سوالات کلیدی زیر با استفاده از آمار است:

- آیا تفاوت امتیازات ثبت‌شده بین دو تیم الف و ب، **از نظر آماری معنادار** است یا ناشی از شانس؟
- با اطمینان ۹۵٪، میانگین ریباندهای یک بازیکن خاص در چه بازه‌ای قرار دارد؟
- اگر تفاوتی وجود دارد، **اندازه اثر (Effect Size)** آن چقدر است؟ (آیا تأثیر آن قابل‌توجه است یا ناچیز؟)
- چه همبستگی‌هایی بین شاخص‌هایی مانند `کارایی بازیکن (PER)` و `درصد پرتاب‌های ۳ امتیازی` وجود دارد؟

تمامی کدها در محیط **Jupyter Notebook** پیاده‌سازی شده‌اند تا فرآیند تحلیل کاملاً شفاف، گام‌به‌گام و قابل بازتولید باشد.

## ⚡ ویژگی‌های کلیدی

- ✅ **آزمون‌های فرضیه پیشرفته**: اجرای خودکار T-test (دو نمونه‌ای مستقل و وابسته)، ANOVA یک‌طرفه و آزمون‌های ناپارامتریک نظیر من-ویتنی.
- ✅ **برآورد بازه اطمینان**: محاسبه بازه اطمینان ۹۰٪، ۹۵٪ و ۹۹٪ برای شاخص‌های کلیدی عملکردی.
- ✅ **سنجش اندازه اثر**: فراتر از p-value رفتن و محاسبه شاخص‌های `Cohen's d` و `Eta-squared` برای تعیین میزان تأثیرگذاری واقعی متغیرها.
- ✅ **تحلیل همبستگی**: شناسایی روابط پنهان بین متغیرها با استفاده از ماتریس همبستگی و نمودارهای پراکندگی.
- ✅ **گزارش‌دهی خودکار**: تولید خروجی‌های آماری به صورت جداول خلاصه و نمودارهای باکیفیت برای ارائه در مقالات یا داشبوردها.

## 🛠️ نصب و راه‌اندازی

برای اجرای این پروژه در محیط محلی خود، مراحل زیر را به ترتیب دنبال کنید:

1. **کلون کردن مخزن** (در صورتی که کل پروژه را کلون نکرده‌اید):
   ```bash
   git clone https://github.com/LiamTaghizadeh/Basketball_analyse.git
   cd Basketball_analyse/src/Phase3-Statistical\ Analysis
   ```

2. **ایجاد محیط مجاز (اختیاری اما پیشنهادی)**:
   ```bash
   python -m venv venv
   source venv/bin/activate  # در لینوکس/مک
   .\venv\Scripts\activate   # در ویندوز
   ```

3. **نصب پیش‌نیازها**:
   ```bash
   pip install -r requirements.txt
   ```
   > اگر فایل `requirements.txt` موجود نیست، کتابخانه‌های اصلی را با دستور زیر نصب کنید:
   ```bash
   pip install numpy pandas scipy statsmodels matplotlib seaborn jupyter
   ```

4. **اجرای Jupyter**:
   ```bash
   jupyter notebook
   ```
   سپس پوشه `Notebooks` را باز کرده و فایل‌های `.ipynb` را به ترتیب شماره اجرا کنید.

## 📦 تشریح ماژول‌ها

ماژول‌های پایتون در پوشه `src/` به شرح زیر عمل می‌کنند:

| فایل | کاربرد |
| :--- | :--- |
| **`data_loader.py`** | بارگذاری فایل‌های CSV، حذف رکوردهای پرت (Outlier) بر اساس میانه و چارک‌ها، و تبدیل داده‌ها به دیتافریم استاندارد. |
| **`statistical_tests.py`** | شامل توابع `t_test_analysis`، `anova_one_way` و `mann_whitney_test` که خروجی آن‌ها به صورت دیکشنری حاوی آماره آزمون، درجه آزادی و p-value است. |
| **`visualizer.py`** | ترسیم نمودارهای باکیفیت با استفاده از کتابخانه `Seaborn`؛ شامل رسم هیستوگرام همراه با منحنی چگالی، نمودار جعبه‌ای برای مقایسه تیم‌ها و نمودار Q-Q برای بررسی نرمال بودن داده‌ها. |

## 📊 خروجی‌ها و مصورسازی

خروجی‌های این فاز به دو دسته تقسیم می‌شوند:

- **جداول آماری**: شامل خلاصه شاخص‌های مرکزی، مقادیر P-value و فاصله اطمینان هر متغیر که در خود نوت‌بوک به صورت DataFrame نمایش داده می‌شوند.
- **نمودارهای تحلیلی**: کلیه نمودارهای تولید شده در پوشه `outputs/figures/` ذخیره می‌شوند که شامل موارد زیر است:
  - مقایسه توزیع امتیازات تیم‌ها با نمودار Violin Plot.
  - ماتریس همبستگی (Correlation Matrix) بین شاخص‌های کلیدی.
  - نمودار باقیمانده‌ها (Residual Plot) برای ارزیابی مدل رگرسیون.
